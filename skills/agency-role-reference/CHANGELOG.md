# Changelog

## 1.0.0 (2026-05-14)

- Initial release.

### What this skill is

A pointer index of ~185 abstract role templates from the upstream [`msitarzewski/agency-agents`](https://github.com/msitarzewski/agency-agents) library (MIT). Each entry in `references/index.md` names a role, gives its one-line description, and links to the upstream raw URL.

The skill ships **no role bodies** — only the index. Consumers (typically `workspace-ceo` during the "Create local" hiring branch) fetch the upstream file on demand, then specialize it for the active mission before writing the resulting `local-agents/<name>/AGENTS.md`.

### Design rationale

- **Reference, not catalog.** The upstream templates are abstract role descriptions, not ready-to-spawn agents. Installing them directly would dilute the catalog and skip the specialization step that makes the resulting hire actually fit the mission.
- **Pointers, not a fork.** Zero redistribution overhead; upstream improvements flow to consumers automatically; license attribution stays simple.
- **Static index.** `references/index.md` is regenerated by hand (walk upstream's tree, pull frontmatter, rewrite the table) — no build tooling lives in this catalog repo. See the SKILL.md "Refreshing the index" section for the recipe.
